2. PROFILE COMPLETION SYSTEM
==================================================

Add a real dynamic Profile Completion percentage.

Do NOT hardcode 83%.

Calculate completion based on important wholesale pharmacy profile fields.

Suggested required fields:

- full_name / owner_name
- business_name
- phone
- email
- address
- GST number
- Drug License number

Example logic:

const profileFields = [
  profile.owner_name,
  profile.business_name,
  profile.phone,
  profile.email,
  profile.address,
  profile.gst_number,
  profile.drug_license_number,
];

const completedFields = profileFields.filter(
  value => value && value.trim() !== ''
).length;

const profileCompletion = Math.round(
  (completedFields / profileFields.length) * 100
);

Display:

Profile Completion: 83%

Add a smooth green progress bar below it.

If profile completion is below 100%, tapping the completion area should open Edit Profile.
